# Troca de Imagem - React (Vite)

Projeto convertido para React com Vite.

Como rodar (agora o app está na raiz do repositório):

1. Abra um terminal na raiz do projeto:

   cd /path/para/o/projeto

2. Instale dependências (se ainda não instalou):

   npm install

3. Rode o servidor de desenvolvimento:

   npm run dev

Observações:
- As imagens estão em `src/assets/images/`.
- Existem dois backups criados antes das limpezas: `backup_before_cleanup_20260101112403.tar.gz` e `backup_react_app_20260101112629.tar.gz`.
- Se quiser que eu configure testes automáticos ou CI, posso adicionar (Jest + React Testing Library).
