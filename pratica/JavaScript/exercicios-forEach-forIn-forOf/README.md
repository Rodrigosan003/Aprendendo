# 📚 Exercícios - forEach, for...in e for...of

---

# ✅ forEach (Arrays)

- [ ] 01. Percorra um array de números e mostre cada número no console.
- [ ] 02. Percorra um array de nomes e mostre cada nome.
- [ ] 03. Mostre o índice e o valor de cada elemento do array.
- [ ] 04. Multiplique todos os números de um array por 2 e exiba o resultado.
- [ ] 05. Some todos os números de um array usando uma variável acumuladora.
- [ ] 06. Exiba apenas os números maiores que 10.
- [ ] 07. Exiba apenas as palavras com mais de 5 letras.
- [ ] 08. Conte quantos números pares existem em um array.
- [ ] 09. Crie um novo array contendo os quadrados dos números utilizando o forEach.
- [ ] 10. Dado um array de objetos representando produtos (nome e preço), mostre uma mensagem no formato: "Notebook custa R$3500".

---

# ✅ for...in (Objetos)

- [ ] 01. Mostre todas as chaves de um objeto.
- [ ] 02. Mostre todos os valores de um objeto.
- [ ] 03. Mostre cada propriedade no formato "chave: valor".
- [ ] 04. Conte quantas propriedades existem no objeto.
- [ ] 05. Exiba apenas as propriedades cujo valor seja do tipo número.
- [ ] 06. Some todos os valores numéricos de um objeto.
- [ ] 07. Exiba apenas as propriedades cujo valor seja uma string.
- [ ] 08. Crie um objeto representando um carro e percorra todas as propriedades.
- [ ] 09. Verifique se existe uma propriedade cujo valor seja `true`.
- [ ] 10. Crie um objeto representando um aluno e exiba uma frase para cada propriedade, por exemplo: "A propriedade nome possui o valor Rodrigo".

---

# ✅ for...of (Iteráveis)

- [ ] 01. Percorra um array de números e exiba cada elemento.
- [ ] 02. Percorra um array de nomes e mostre todos no console.
- [ ] 03. Some todos os números de um array.
- [ ] 04. Exiba apenas os números maiores que 50.
- [ ] 05. Conte quantos números são pares em um array.
- [ ] 06. Percorra uma string e mostre cada letra em uma linha.
- [ ] 07. Conte quantas letras possui uma palavra usando o for...of.
- [ ] 08. Conte quantas vogais existem em uma string.
- [ ] 09. Percorra um array de frutas e interrompa o laço quando encontrar "banana".
- [ ] 10. Percorra um array de objetos representando pessoas e exiba apenas o nome de cada uma.

---

# 🏆 Desafios Extras

- [ ] Crie um programa que utilize `forEach` para calcular a média das notas de um array.
- [ ] Crie um objeto com informações de um livro e utilize `for...in` para imprimir todas as informações.
- [ ] Percorra uma frase utilizando `for...of` e conte quantos espaços em branco existem.
- [ ] Compare a saída produzida por `forEach` e `for...of` utilizando o mesmo array.
- [ ] Pesquise e explique, com suas próprias palavras, quando é mais indicado utilizar `forEach`, `for...in` e `for...of`.
