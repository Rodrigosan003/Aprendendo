# Checklist de Exercícios MySQL

## Nível Basico

- [ ] 01. Crie uma tabela simples com duas colunas: ID (int) e Nome (varchar).
- [ ] 02. Insira 5 registros na tabela criada no exercício anterior.
- [ ] 03. Selecione todos os registros da tabela criada no exercício 1.
- [ ] 04. Atualize o nome de um dos registros da tabela criada no exercício 1.
- [ ] 05. Delete um registro da tabela criada no exercício 1.
- [ ] 06. Selecione todos os registros onde o ID seja maior que 3 da tabela criada no exercício 1.
- [ ] 07. Crie uma tabela nova que faça referência à tabela criada no exercício 1 usando uma chave estrangeira.
- [ ] 08. Selecione todos os registros de ambas as tabelas usando um join simples.
- [ ] 09. Faça uma consulta que retorne a contagem de registros na tabela criada no exercício 1.
- [ ] 10. Crie um procedimento armazenado que insira um registro na tabela criada no exercício 1.

## Nível Medio

- [ ] 01. Crie uma consulta que retorne os registros únicos da coluna Nome da tabela criada no exercício 1.
- [ ] 02. Crie uma consulta que retorne a média dos IDs da tabela criada no exercício 1.
- [ ] 03. Crie uma consulta que retorne os registros ordenados pelo Nome de forma descendente.
- [ ] 04. Crie uma consulta que utilize uma subconsulta para retornar os IDs onde o Nome comece com 'A'.
- [ ] 05. Crie uma consulta que utilize joins para retornar registros de três tabelas relacionadas.
- [ ] 06. Atualize todos os registros de uma tabela usando uma condição complexa.
- [ ] 07. Crie um índice na coluna ID da tabela criada no exercício 1.
- [ ] 08. Utilize transações para garantir a consistência de operações em duas tabelas ao mesmo tempo.
- [ ] 09. Crie um trigger que atualize uma terceira tabela sempre que ocorrer uma inserção na tabela criada no exercício 1.
- [ ] 10. Crie uma função que retorne a soma de todos os IDs da tabela criada no exercício 1.

## Nível Avancado

- [ ] 01. Crie uma consulta recursiva que liste todos os registros de uma tabela hierárquica.
- [ ] 02. Utilize procedimentos armazenados para automatizar a geração de dados em massa para a tabela criada no exercício 1.
- [ ] 03. Utilize particionamento para otimizar consultas na tabela criada no exercício 1.
- [ ] 04. Crie uma consulta que utilize índices espaciais para encontrar registros próximos a uma localização geográfica.
- [ ] 05. Crie uma consulta que utilize funções analíticas para calcular uma média móvel dos IDs da tabela criada no exercício 1.
- [ ] 06. Crie uma visão que agregue dados de múltiplas tabelas relacionadas à tabela criada no exercício 1.
- [ ] 07. Crie um procedimento armazenado que faça a replicação assíncrona de dados entre duas instâncias MySQL.
- [ ] 08. Utilize técnicas de segurança avançada para proteger dados sensíveis na tabela criada no exercício 1.
- [ ] 09. Implemente controle de versão de esquema para garantir que alterações na estrutura da tabela sejam rastreadas.
- [ ] 10. Crie uma consulta que utilize JSON functions para manipular dados estruturados na tabela criada no exercício 1.
