# ✅ Checklist de Exercícios Node.js

## Basico

- [ ] 01. Criar um servidor HTTP simples que responda 'Olá, mundo!'.
- [ ] 02. Ler e escrever arquivos .txt usando o módulo fs.
- [ ] 03. Criar um script de linha de comando que imprime seu nome (usando process.argv).
- [ ] 04. Listar arquivos de um diretório com fs.readdir.
- [ ] 05. Criar um contador de visitas que salva o número de acessos em um arquivo.
- [ ] 06. Formatar data e hora atual no terminal com Date.
- [ ] 07. Criar um módulo próprio que exporta uma função e importá-lo em outro arquivo.
- [ ] 08. Ler um arquivo JSON e exibir os dados no terminal.
- [ ] 09. Criar um servidor que retorna JSON (ex: dados de uma pessoa).
- [ ] 10. Usar npm init e instalar um pacote simples como moment.

## Medio

- [ ] 01. Criar uma API REST simples com Express (GET/POST de usuários).
- [ ] 02. Salvar dados enviados via POST em um arquivo JSON.
- [ ] 03. Fazer validação de entrada de dados com express-validator ou Joi.
- [ ] 04. Utilizar middleware no Express para logar requisições.
- [ ] 05. Criar rotas com parâmetros e query strings.
- [ ] 06. Implementar sistema de autenticação básico com token JWT.
- [ ] 07. Integrar com banco de dados MongoDB usando mongoose.
- [ ] 08. Criar um sistema de CRUD (ex: tarefas, usuários ou produtos).
- [ ] 09. Trabalhar com variáveis de ambiente (.env) com dotenv.
- [ ] 10. Implementar tratamento de erros global para rotas inválidas ou erros do servidor.

## Avancado

- [ ] 01. Criar uma API com autenticação completa (login, logout, refresh token).
- [ ] 02. Cachear requisições com Redis para melhorar desempenho.
- [ ] 03. Testar a aplicação com Jest ou Mocha/Chai.
- [ ] 04. Implementar uploads de arquivos com multer.
- [ ] 05. Criar WebSockets em tempo real com Socket.io (ex: chat).
- [ ] 06. Aplicar arquitetura em camadas (Controller, Service, Repository).
- [ ] 07. Fazer deploy da API usando Docker e Docker Compose.
- [ ] 08. Criar filas de tarefas assíncronas com Bull e Redis.
- [ ] 09. Rate limiting com express-rate-limit para proteção contra DDoS.
- [ ] 10. Integração com serviços externos via API (ex: API do GitHub, Twilio, SendGrid).
