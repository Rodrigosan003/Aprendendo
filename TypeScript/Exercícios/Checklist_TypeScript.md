# Checklist de Exercícios TypeScript

## BÁSICO
- [ ] Tipar variáveis primitivas: Crie variáveis com os tipos string, number, boolean e null.
- [ ] Função com tipagem explícita: Crie uma função soma(a: number, b: number): number que retorna a soma.
- [ ] Objeto com interface simples: Crie uma interface Pessoa e uma função que a utiliza.
- [ ] Array tipado de strings: Crie uma função que imprime todos os nomes de um array de strings.
- [ ] Tupla simples: Crie uma tupla [string, number] com nome e idade.
- [ ] Enum simples: Enum com dias da semana e mensagens personalizadas.
- [ ] Union Types: Função que aceita string ou number e trata cada caso.
- [ ] Type Alias: Crie um type Status = 'ativo' | 'inativo' e use num objeto.
- [ ] Type Assertion: Force um tipo em uma variável any para string.
- [ ] Parâmetro opcional: Função que trata parâmetro opcional com mensagem diferente.

## INTERMEDIÁRIO
- [ ] Interface com propriedades opcionais e readonly.
- [ ] Função Genérica: retornarPrimeiro<T>(arr: T[]): T.
- [ ] Classe com métodos e construtor: Carro com ligar() e desligar().
- [ ] Interface estendida: Funcionario estende Pessoa.
- [ ] Intersection Types: Combine dois tipos com &.
- [ ] Literal Types: Aceitar apenas 'admin' ou 'user'.
- [ ] Manipulação de tipos com keyof e typeof.
- [ ] Classe com interface: ICalculadora implementada.
- [ ] Função que retorna never: lança erro.
- [ ] Enum personalizado: Com valores de string ou número.

## AVANÇADO
- [ ] Generic com restrição (extends).
- [ ] Mapped Types: Torne todas propriedades readonly.
- [ ] Utility Types: Partial, Pick e Omit em Usuario.
- [ ] Tipos condicionais com T extends U ? X : Y.
- [ ] Classe genérica: Repositorio<T> com métodos.
- [ ] Módulos e Namespaces com import/export.
- [ ] Decorators experimentais: log de métodos.
- [ ] Inferência com infer para tipo de retorno.
- [ ] Tipos recursivos: menu com submenus.
- [ ] Type Guards: Validação com typeof e instanceof.
