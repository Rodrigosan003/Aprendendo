# 🚀 Laboratório de Estudos e Práticas

> *"Aprender fazendo."*

Bem-vindo ao meu laboratório de estudos!

Este repositório reúne todos os projetos, exercícios, desafios e experimentos que desenvolvo durante minha jornada como desenvolvedor **Full Stack**. Cada pasta representa um passo da minha evolução, desde conceitos fundamentais até projetos mais completos.

Meu objetivo não é apenas fazer algo funcionar, mas compreender profundamente como cada tecnologia opera. Por isso, grande parte dos projetos é construída sem depender de abstrações desnecessárias, priorizando o entendimento dos fundamentos.

Além da programação, tenho grande interesse por **UI/UX**, organização de código, automação e ambiente Linux. Acredito que unir criatividade com lógica é uma das melhores formas de criar soluções de qualidade.
